Seajure is the Seattle Clojure Group. Yes, it's a silly name.

  • Time: 7:00 pm on the first Thursday of each month
  • Location: Offices of Secure One, 1518 1st Ave S, Seattle
  • Mailing list
  • Meetup.com Web Site

We generally start the meetings off with half an hour or so of discussion and then dive into a hackfest. We try to find a bite-sized problem and do some swarm-coding to collaborate on a solution together. Bring a laptop!

Lately we have been using Syme to facilitate collaboration. If you ensure your SSH public key is on your GitHub account and get added to the Seajure organization before the meeting it helps things go more smoothly.

We have a wiki where we brainstorm future project ideas.
